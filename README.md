1. What is JSX, and why is it used?

Answer:
JSX means JavaScript XML. It allows writing HTML code inside JavaScript.

2. What is the difference between State and Props?

Answer:

Props: Used to pass data from one component to another. They are read-only.

State: Used to manage data within a component. It can be read and updated.

3. What is the useState hook, and how does it work?

Answer:
useState hook is used in functional components to store and update data dynamically.

state: holds the current value.

setState: updates the value.

5. How is event handling done in React?

Answer:
Event handlers are connected to functions via props. Common events include:
onClick,onChange,onSubmit
These events are linked to functions to perform actions when triggered
